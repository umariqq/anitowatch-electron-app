# AniToWatch - Anime & Manga Tracking Desktop Application

![AniToWatch](https://img.shields.io/badge/Electron-Desktop_App-47848F?style=for-the-badge&logo=electron)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-F7DF1E?style=for-the-badge&logo=javascript)
![Jikan API](https://img.shields.io/badge/Jikan-API-2E51A2?style=for-the-badge)

A modern desktop application for anime and manga enthusiasts built with **Electron.js**, featuring real-time data from **Jikan API** (Unofficial MyAnimeList API). This application allows users to discover, track, and manage their favorite anime, manga, and characters.
